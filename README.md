# 📊 DIS Client Scoring System

Интеллектуальная система скоринга клиентов ДИС для оценки **риска ухода клиентов** и **вероятности их возврата**.

Система поддерживает два режима анализа:

- 🤖 ML-скоринг на основе CatBoost;
- ⚙️ Полностью настраиваемый ручной скоринг.

---

## ✨ Основные возможности

### 🤖 ML-скоринг

Использование обученной модели CatBoost для автоматической оценки клиентов.

**Поддерживаемые сценарии**

#### Сопровождаемые клиенты

- оценка вероятности ухода;
- определение уровня риска;
- выявление основных факторов риска;
- визуализация распределения клиентов.

#### Ушедшие клиенты

- оценка вероятности возврата;
- определение уровня возвращаемости;
- анализ причин отключения;
- ранжирование клиентов по перспективности возврата.

---

### ⚙️ Ручной скоринг

Позволяет построить собственную модель оценки без переобучения ML-модели.

Возможности конструктора:

- изменение списка признаков;
- добавление новых признаков;
- удаление признаков;
- изменение весов;
- выбор формулы расчёта;
- выбор типа признака;
- автоматическая нормализация весов;
- сохранение конфигурации.

## 💻 Использование

1. Выберите режим скоринга.
2. Выберите тип клиентов.
3. Загрузите Excel-файл.
4. При необходимости настройте ручной скоринг.
5. Запустите расчёт.
6. Проанализируйте результаты.
7. Скачайте итоговый Excel.
