\# 📊 Client Scoring System



Система интеллектуального скоринга клиентов ДИС, предназначенная для оценки:



\- риска ухода сопровождаемых клиентов;

\- вероятности возврата ушедших клиентов;

\- ручной настройки формулы скоринга без изменения программного кода.



Проект объединяет ML-модель, ручной конструктор скоринга и удобный веб-интерфейс для анализа клиентской базы.



\---



\# Возможности



\## ML-скоринг



Использование заранее обученной модели CatBoost.



Поддерживаются два режима:



\- Сопровождаемые клиенты

&#x20;   - вероятность ухода

&#x20;   - уровень риска

&#x20;   - ключевые факторы риска



\- Ушедшие клиенты

&#x20;   - вероятность возврата

&#x20;   - уровень возвращаемости

&#x20;   - анализ причин отключения



\---



\## Ручной скоринг



Позволяет полностью изменить методику расчета без переобучения модели.



Возможности:



\- изменение списка признаков;

\- добавление новых признаков;

\- удаление ненужных признаков;

\- изменение весов;

\- выбор формулы расчета;

\- выбор типа признака;

\- автоматическая корректировка весов;

\- сохранение конфигурации.



\---



\## Работа с Excel



Поддерживается:



\- загрузка Excel-файлов;

\- автоматическая проверка структуры;

\- очистка данных;

\- расчёт скоринга;

\- экспорт результатов обратно в Excel.

После расчета отображаются:

\# API



\## ML



```

POST /score/churn

```



Расчет риска ухода активных клиентов.



\---



```

POST /score/winback

```



Расчет вероятности возврата ушедших клиентов.



\---



\## Manual



```

POST /score/manual

```



Ручной скоринг.



\---



\## Features



```

GET /features

```



Получить список признаков.



```

POST /features

```



Добавить признак.



```

PUT /features/{name}

```



Изменить признак.



```

DELETE /features/{name}

```



Удалить признак.
